Mes Projets

Alert MNS

Application de mesagerie

Projet fil rouge de la formation Développeur Web 2022-2023 Développement d'une application de messagerie d'entreprise.


Technologies et outils utilisés :

  • Développement Front-end : Utilisation de HTML, CSS et JavaScript
  • Développement Back-end : Utilisation de PHP et SQL avec des mesures de sécurité contre les failles XSS et les injections SQL
  • Bases de données : Modélisation avec JMerise (MCD), MySQL
  • Interaction Front/Back : Communication via des requêtes fetch en JSON
  • Hébergement et administration serveur : Déploiement sur OVH avec configuration NGINX
  • Organisation et suivi de projet : Méthodologie agile avec Trello
  • UI/UX et design : Création de maquettes sur Canva, élaboration d'une charte graphique et adaptation responsive
  • Gestion du code source : Versioning avec Git

Infinite Quiz

Application de Quiz

Projet fil rouge de la formation Bachelor Développeur Fullstack 2023-2024


L’objectif principal est de développer un outil de gamification permettant d’évaluer les connaissances et compétences des collaborateurs, tout en facilitant l’intégration d’Icebreakers lors de réunions ou séminaires. L’application est conçue en mode SaaS.


Technologies et outils utilisés :

  • Développement Front-end : Angular
  • Développement Back-end : Symfony, Api Platform
  • Bases de données : Modélisation avec JMerise (MCD), MySQL
  • Interaction Front/Back : Communication via l'API avec des requêtes FETCH
  • Intégration continue (CI/CD) : Mise en place de l'intégration continue avec GitHub Actions
  • Tests : Réalisation de tests unitaires et fonctionnels pour assurer la qualité du code
  • Gestion de projet : Modélisation UML, gestion des risques, planification des tâches avec GANTT, et méthodologie agile pour le suivi de l'avancement Kanban avec Trello
  • UI/UX et design : Création de maquettes sur Figma, élaboration d'une charte graphique et adaptation responsive
  • Gestion du code source : Versioning avec Git
